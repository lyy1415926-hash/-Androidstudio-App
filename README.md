# -Androidstudio-App
一、项目背景与目标
通过开发一个校园信息助手应用，帮助学生便捷查看校园新闻、课程表与天气信息，熟悉安卓应用的基本结构和网络通信技术。
二、功能模块要求
基础功能：
1.校园新闻展示（从JSON或API获取数据）
2.课程表模块（支持增删改查）
3.天气查询功能（调用天气API）
扩展功能：
1.登录注册与个性化界面
2.夜间模式、消息提醒
3.新闻收藏与搜索功能
三、技术要求与建议
使用 Activity、RecyclerView、Intent 等基础组件
数据存储采用 SQLite/Room
网络通信使用 OkHttp 或 Retrofit
UI设计规范简洁，采用 Material Design 风格
